# My Expense Tracker App

## Overview

My Expense Tracker is an Android application developed using Java and SQLite. The app helps users manage daily expenses by recording, updating, and tracking transactions.

## Features

* Add new expenses
* Edit existing expenses
* Delete expense records
* Store data using SQLite database
* User-friendly Android interface

## Technologies Used

* Java
* Android Studio
* SQLite
* XML

## Learning Outcomes

* Android application development
* SQLite database integration
* CRUD operations
* UI design using XML
